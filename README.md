# todoApp

 

# Requirement

- Python 3.5 or higer

- Django 2.2.5

- Django-filter 2.2.0

- Django REST framework 3.10.3

- Django REST framework-jwt 1.11.0

- Node v8.x or higher

- NPM v5.x or higher

- Angular 8.3.6

- Angular CLI version 8.3.6

- bootstrap 4.3.1

# Installation

pip install django

pip install djangorestframework

pip install djangorestframework-jwt

pip install django-filter

npm install
